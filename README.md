# Seabrook Plumbing & Heating Website

Static landing page for `seabrookplumbing.ca`.

## Upload To GitHub

Upload the contents of this folder to the GitHub repository root:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`
- `favicon.png`
- `favicon-96x96.png`
- `apple-touch-icon.png`
- `robots.txt`
- `sitemap.xml`
- `CNAME`
- `404.html`

The `CNAME` file is set to `seabrookplumbing.ca`.

## Notes

- The phone number is pulled from the supplied header artwork: `705-206-1696`.
- The dark header/background color matches the supplied header artwork.
- The page is static and can be hosted on GitHub Pages.
